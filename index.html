Revelo UI Gen example

Prompt

I run a little workshop in Turin where I restore old Italian espresso machines by hand, one at a time. Faema, Gaggia, La Cimbali, that kind of thing. The shop is called Crema Vecchia. I want a website for it, something that feels properly high-end, like the design studio sites that win awards, with smooth animations and beautiful typography. Nothing corporate or templated looking. Use a nice font and make it feel crafted.

Results

Claude Opus 4.8 Redesign

UI Notes

The original design had very beautiful typography, so I kept it similar. I just got a more modern sans serif font for the body text, so it doesn't feel too vintage. On the colors, it was using Claude standard color palette, which is being overused around the web. So I suggested a lighter and more original color palette of light purple, bright blue and white, so the real content of the website can shine and it looks fresher. I created some images of the user and his machines with AI, so it looks more realistic than the illustrations of the original site. Also, added motion design in the header, text and in the main pieces of content, so the site is more interactive and fun to use.
